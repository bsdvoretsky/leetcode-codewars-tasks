# Evaluate mathematical expression

https://www.codewars.com/kata/52a78825cdfc2cfc87000005/javascript

Given a mathematical expression as a string you must return the result as a number.

1-1    // 0
1 -1   // 0
1- 1   // 0
1 - 1  // 0
1- -1  // 2
1 - -1 // 2
1--1   // 2

6 + -(4)   // 2
6 + -( -4) // 10
